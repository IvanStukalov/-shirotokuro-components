# Components for my own projects

#### Install

`
npm install @shirotokuro/components
`

#### Usage

```javascript
colors = {
  backgroundPrimary: "#1D1D1B",
  backgroundSecondary: "#4E5154",

  secondary: "#2BC8D2",
  danger: "#D22B53",
```
